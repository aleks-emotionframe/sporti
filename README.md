# Sporti – Dein Sportverein. Deine App.

Sportverein-Management App für Trainings, Spiele, Turniere & Events.

## 🚀 Live Demo
**[sporti.aleks-emotionframe.github.io/sporti](https://aleks-emotionframe.github.io/sporti)**

## Features
- 📅 An-/Abmeldung für Trainings, Spiele, Turniere, Events
- 🏆 Punktesystem & Sportler/in des Jahres
- 👥 Mitgliederverwaltung (Spieler, Trainer, Vorstand, Admin)
- 👨‍👩‍👧 Eltern-Kind Verknüpfung mit Punkteübertragung
- 🤝 Helfereinsätze mit Punktevergabe
- 💬 Team-Chat
- 💰 Finanzen & Bussen-System
- 🏥 Abwesenheiten & Verletzungs-Tracking
- 🚗 Fahrgemeinschaften
- 🔔 Push-Benachrichtigungen (PWA)
- 🔐 Rollenbasierte Berechtigungen

## Tech Stack
- **Prototyp:** HTML/CSS/JS + React (CDN)
- **Produktion:** PHP 8+ / MySQL / Hostpoint
